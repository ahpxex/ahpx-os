# ahpx-os

A Windows-XP-inspired personal web OS built with React, Vite, TanStack Router, and Jotai.

## Stack

- Frontend: React 19 + Vite + TanStack Router + Jotai
- Storage: Browser `localStorage`

## Local setup

1. Install dependencies:
   - `bun install`
2. Start the app:
   - `bun run dev`

The app runs on `http://localhost:5173`.

## Notes

- Profiles and blog posts are stored locally in your browser.
- Deleting site data or clearing `localStorage` resets the seeded content.
- Build with `bun run build` and preview with `bun run preview`.
